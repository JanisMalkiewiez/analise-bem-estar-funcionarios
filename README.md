# 🎯 Análise de Bem-Estar dos Funcionários com Excel

Este é o meu **primeiro projeto em Análise de Dados**, que foi criado do zero por mim, com dedicação e olhar analítico, buscando traduzir os dados em informações claras e acionáveis.

  
A proposta foi analisar indicadores de bem-estar dentro de uma organização, utilizando Excel como principal ferramenta de visualização e exploração dos dados.

A base contém informações de 100 funcionários avaliados em seis dimensões principais:

- **Estresse**
- **Qualidade do Sono**
- **Produtividade**
- **Satisfação**
- **Faixa de Satisfação**
- **Nível de Estresse**


Com esse projeto, busquei transformar dados em **insights claros e acessíveis**, por meio de dashboards intuitivos e análises estatísticas completas.

## 💡 Objetivo do Projeto

Avaliar de forma visual e estatística como os funcionários estão se sentindo em relação a estresse, sono, produtividade e satisfação, ajudando na tomada de decisões estratégicas de bem-estar no ambiente corporativo.

---

## 🧾 O que foi feito

- ✅ Limpeza e organização da base de dados
- 📊 Criação de **Tabelas Dinâmicas** para análises segmentadas
- 📈 Construção de **Dashboards** com gráficos de colunas, barras, dispersão e pizza
- 📉 Criação de **Histogramas** para visualizar a distribuição das variáveis
- 📌 Aplicação de **Estatísticas Descritivas** (média, desvio padrão, variância, assimetria, curtose)
- 🏅 Geração de **Rankings** por funcionário e por departamento
- ❓ Abas dedicadas para **Perguntas e Respostas** e **Observações Analíticas**
- 📚 Aplicação de conceitos estatísticos para avaliar o bem-estar geral da equipe
